# 🌐 Assignment 3: Web Interface with MetaMask Integration

## 📌 Brief Description
In this assignment, a simple web-based decentralized application (DApp) is developed to interact with the deployed **ProductTransaction smart contract**.  
Users can connect their MetaMask wallet and perform blockchain transactions like adding and retrieving product details.

---

## 🛠️ Tech Stack Used
- HTML  
- CSS  
- JavaScript  
- Ethers.js  
- MetaMask  

---

## 🚀 Features Implemented

### 🔹 Connect Wallet
- Users can connect their MetaMask wallet  
- Displays connected account address  

### 🔹 Add Product
- User enters Product ID and Name  
- Transaction is sent to blockchain  

### 🔹 Get Product
- Fetch product details from blockchain  
- Displays product name and owner address  

---

## 🚀 How to Run

1. Open project folder in VS Code  
2. Run using **Live Server**  
3. Open in browser (http://localhost)  
4. Click **Connect Wallet**  
5. Approve MetaMask connection  
6. Use features:
   - Add Product  
   - Get Product
